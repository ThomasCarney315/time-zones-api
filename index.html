Time Zones API

-27 Standard time zones to keep it simple.

-All Universal Coordinated Time(UTC) referenced.

-CORS = Works on your local machine.

-Served as JSON

How to use this api

Find a single time zone object:

https://adorable-toad-bikini.cyclic.app/api/tz/<gmt>

Get everything:

https://adorable-toad-bikini.cyclic.app/api

Get simple list of time zones:

https://adorable-toad-bikini.cyclic.app/api/list

Get simple list of time zones with UTC +- reference

https://adorable-toad-bikini.cyclic.app/api/utclist

Get simple list of time zones with UTC +- & and details

https://adorable-toad-bikini.cyclic.app/api/detailedlist
